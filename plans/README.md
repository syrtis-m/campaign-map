## Point-crawl connections (the requested feature)

Split into 004 (make connections declared in note frontmatter render as themed
lines, compatible with every map style) and 005 (create/remove them from the
map UI). The design is canon-native: a connection is a `connections:` list in a
location note's frontmatter, resolved at reconcile time into line features ‚Äî
same "canon = notes" model as everything else, so links sync, survive rename,
and vanish when an endpoint is deleted. Straight undirected lines in v1;
curved/waypointed paths, directional arrowheads, and travel-time edge labels are
scoped as follow-ups inside those plans.
